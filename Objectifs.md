# Objectifs :
-  Programmer l'encodage/décodage pour les 4 algos suivants : Chiffre de César, Substitution monoalphabétique, Scytale et chiffre de Vigenère.
-  Laisser la possibilitée a l'utilisateur de gérer l'entrée comme bon lui semble ("supprimer les espaces ou les caractères spéciaux, chiffrer un texte écrit dans un fichier .txt").
-  Généraliser le code de César ou le chiffre de Vigenère à des alphabets plus grands
-  Programmer une attaque pour chacun des 4 algos.

# Fonctions : 
- Code de césar : chiffrage, déchiffrage, crackage (bruteforce), choix de l'alphabet. (danielli)
- Substitution monoalphabétique : chiffrage, déchiffrage, crackage. (aminata)
- Scytale : chiffrage, déchiffrage, crackage. (gaspard)
- Vigenère : chiffrage, déchiffrage, crackage, choix de l'alphabet (done).
- Une fonction qui réccupère l'information dans un fichier texte. (gaspard)
- Une fonction qui permet de se débarrasser ou non des caractères spéciaux/espaces. (aminata)

# Json (rémy) :
- Mettre les dico dans un json qu'on appellera
- Mettre les IC par langue dans un dico
- Si on veut généraliser tout le code pour tout alphabet dans le monde, il faudra faire le dico contenant : alphabet, IC, len(alphabet), probabilitées de présences des lettres (normalement ca suffit a généraliser le code a des alphabets non latins) + modifs dans le code pour rendre tt ca fonctionnel.
