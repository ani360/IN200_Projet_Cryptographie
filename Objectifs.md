# Objectifs :
-  Programmer l'encodage/décodage pour les 4 algos suivants : Chiffre de César, Substitution monoalphabétique, Scytale et chiffre de Vigenère.
-  Laisser la possibilitée a l'utilisateur de gérer l'entrée comme bon lui semble ("supprimer les espaces ou les caractères spéciaux, chiffrer un texte écrit dans un fichier .txt").
-  Généraliser le code de César ou le chiffre de Vigenère à des alphabets plus grands
-  Programmer une attaque pour chacun des 4 algos.

# Fonctions : 
- Code de césar : chiffrage, déchiffrage, crackage (bruteforce), choix de l'alphabet.
- Substitution monoalphabétique : chiffrage, déchiffrage, crackage.
- Scytale : chiffrage, déchiffrage, crackage.
- Vigenère : chiffrage, déchiffrage, crackage, choix de l'alphabet (je m'occupe de cet algo).
- Une fonction qui réccupère l'information dans un fichier texte.
- Une fonction qui permet de se débarrasser ou non des caractères spéciaux/espaces.

# Bonus :
- Enigma : chiffrage, déchiffrage, crackage.
